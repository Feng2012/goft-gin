# Goft-Gin
* 在web框架gin的基础上做的脚手架

## 安装
go get -u github.com/shenyisyn/goft-gin@v0.3.0

## 功能说明
 控制器、简易依赖注入、中间件、表达式、任务组件等。
 
 后续功能正在发布中,因此**可能有较大改动**
